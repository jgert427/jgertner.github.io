How do Genetic Algorithms Actually Work?

All genetic algorithms follow this basic format:

Initialization- Create the initial population, this is usually randomly generated and can be any size as needed

Evaluation- Each member, or chromosome as it’s referred to in a genetic algorithm, is then evaluated and given a fitness score. The fitness score is calculated based on how well each chromosome fits to the desired parameter, so better solutions get better scores.

Selection- There are many different ways of determining selection but all selections have the same goal in mind, make the better outcome more likely, therefore only chromosomes with higher scores are retained.

Crossover- Aspects of chromosomes are combined to create new chromosomes. This is just like reproduction in nature and the hope is to produce a new and better chromosome with a higher fitness score.

Mutation- So as to not have only the possibilities contained in the original population random small changes are made

Repeat- With a brand new population the process can repeat until a termination condition is reached

Termination- Termination is determined when no improvements are found over a set number of generations.

Objective Functions- A term used to describe seperate functions in the code of the algorithm that create the processes used for the selection, crossover, and mutation steps. Most algorithms will have multiple ways of doing these steps to optimize the outcome and each algorithm can do them differently. This is what allows genetic algorithms to be so wide spread in thier use in different fields.
