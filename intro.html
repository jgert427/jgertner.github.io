What is a Genetic Algorithm?

Genetic algorithms are a group of algorithms based on evolution. In genetics certain genes are passed on through the generations because they confer advantageous triats for survival. Those with these advantageous genes become more likely to survive and reproduce. A genetic algorithm uses a computational model to mimic this process for the given problem. Instead of genes the algorithm passes down better answers to the given problem. It then interchanges parts of these answers with each other to arrive at the optimal answer faster; adding minor changes along the way to increase variety, like random mutations in nature. Therefore all genetic algorithms are designed with a an optimized goal in mind, given all possible parameters.
