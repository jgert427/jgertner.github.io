What is a Genetic Algorithm?

Genetic algorithms are a group of algorithms based on evolution. In nature evolution favors those with advantageous traits which are then able to reproduce and pass on those traits to further generations. So a genetic algorithm uses a computational model to mimic this process for a given problem. A genetic algorithm is designed with a an optimized goal in mind and given all parameters involved leads to the best solution for the given problem.
